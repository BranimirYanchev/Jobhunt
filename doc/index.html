OVERVIEW

Jobhub is a modern Job board HTML Template. We design Jobhub after understanding of the business to connect employers and candidates. This template suits for any type of Job listings, Job Board & Job Search. Using Jobhub HTML Template you can create a complete fully Responsive job board, job posting recruitment website.

Please note that this is pure HTML5 Template and NOT a WordPress Theme.

The Jobhub - Job board HTML5 website template offers clean & corporate design suitable for modern job search portals. This solution comes with a spacious layout. It is suited for various types of content. From text to images and color, this template handles anything quite well. Also, it is a part of the Jobhub template family. The latter offers a variety of child themes and templates at a single & affordable price. This makes Jobhub a great choice for any website.

Moreover, our HTML solution provides a set of features designed to make your website more sustainable:

  • W3C valid coding;
  • SEO-ready layout;
  • Google Fonts & Google Maps;
  • Cross-browser support;
  • SCSS source files.

Jobhub s built with Bootstrap 5, HTML5, CSS3 & Jquery. It is Responsive, Retina ready & Multi-Device supported. All code is beautifully written & W3C Validated.

This template written by SASS, with 7-1 pattern

We have a dedicated support center for all of your support needs. It includes our Documentation and Ticket system for any questions you have. We usually get back to you within 12-24 hours.

Buy today and start your job board or services listing marketplace.

Import google fonts 

                                @import url("https://fonts.googleapis.com/css2?family=Montserrat:wght@400;500;600;700&family=Open+Sans:wght@400;600;700&display=swap");

Import third party CSS library 

                                /*import vendors*/
                                @import url(vendors/normalize.css);
                                @import url(vendors/bootstrap.min.css);
                                @import url(vendors/uicons-regular-rounded.css);

                                /*import plugins*/

UPDATE

Important Note:

- Backup your website and theme folders completely to make sure you have a restore point in case of any complications.

Update using FTP

  1. Download the latest version of the theme to your device.
  2. Connect to your hosting via FTP software
  3. Upload the theme folder you unzipped to your site folder
  4. You should be asked to confirm overwriting of this folder and its contents. Confirm the overwrite.
